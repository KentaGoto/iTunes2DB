# iTunes2DB

## Description
Store metadata of music files (*.m4a or *.mp3) managed by iTunes in SQLite.   
The items in the table are as follows.  

- ID
- TITLE
- ARTIST
- ALBUM
- TRACK_NUMBER

The following image shows the generated *.db opened with the **VS Code** plugin **SQLite Viewer**.
